## 📖 Sobre o Projeto

Este projeto nasceu com a missão de **conscientizar sobre a importância da gestão eficiente dos recursos naturais** na indústria.

Através de um site interativo em HTML e CSS, buscamos mostrar como **pequenas atitudes e boas práticas industriais** podem gerar grandes impactos positivos na preservação ambiental e na construção de um futuro sustentável.
